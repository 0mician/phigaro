# Phigaro

## Installation

```
# with sudo
python3 install setup.py
```

## Usage
```
phigaro -h                                                                                                                                                                          15:04:32
usage: phigaro [-h] [-w WINDOW_LEN] [-m THRESHOLD_MIN] [-M THRESHOLD_MAX]
               npn_file

Tool for predicting prophages in metagenomes

positional arguments:
  npn_file

optional arguments:
  -h, --help            show this help message and exit
  -w WINDOW_LEN, --window-len WINDOW_LEN
  -m THRESHOLD_MIN, --threshold-min THRESHOLD_MIN
  -M THRESHOLD_MAX, --threshold-max THRESHOLD_MAX
```
